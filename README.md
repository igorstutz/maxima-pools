## 📖 About

Modern marketing website for **Maxima Pools**, a premium fiberglass pools and spas company based in Columbus, OH. The project replaces a legacy presence with a fast, content-driven site built on the latest Next.js stack.

## ✨ Features

- ⚡ Server-side rendered pages with **Next.js 16 App Router**
- 📝 Markdown-driven content via `react-markdown` + `remark-gfm`
- 🎨 Modern UI with **Tailwind CSS 4**
- 🧊 Reusable iconography with **lucide-react**
- 🚀 Optimized for SEO and performance

## 🛠️ Stack

| Layer        | Tech                                                      |
| ------------ | --------------------------------------------------------- |
| Framework    | Next.js 16 (App Router)                                   |
| UI           | React 19 · Tailwind CSS 4 · Lucide React                  |
| Content      | Markdown + `gray-matter` (frontmatter parsing)            |
| Language     | TypeScript 6                                              |
| Tooling      | PostCSS · ESLint                                          |

## 🚀 Running locally

```bash
git clone https://github.com/igorstutz/maxima-pools.git
cd maxima-pools
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## 📦 Build

```bash
npm run build
npm start
```

See [`DEPLOY.md`](./DEPLOY.md) for deployment instructions.
